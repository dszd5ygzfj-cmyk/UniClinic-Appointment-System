# UniClinic-Appointment-System
University Clinic Appointment System prototype supporting SDG 3.

Clinic Appointment System

The University Clinic Appointment System is designed to support SDG 3: Good Health and Well-Being by making healthcare services at the university clinic easier to access. The system allows students to book, view, and cancel clinic appointments online, while also helping doctors and administrators manage schedules more effectively. By digitizing the appointment process, the system helps reduce long waiting times and improves the overall healthcare experience for students.

Overview of SDLC Work

This project was developed using the Software Development Life Cycle (SDLC) to ensure the system was planned, designed, and tested in an organized way.

Planning

During the planning phase, the project goals and product vision were defined. User stories were created to understand how students and administrators would interact with the system, and the basic project structure was prepared.

Design

In the design stage, UML diagrams such as Use Case diagrams and Class diagrams were created to model how the system works. The user interface was also planned to make the system simple and user-friendly.

Testing

Testing was conducted to check whether the system works correctly. Test plans were created, and feedback from users was collected to identify areas that could be improved.

Final Stage

In the final stage, the system prototype was completed. Documentation and the project report were prepared, and the final results of the project were presented.

Key Features

Register

Students can create a personal account to access the system and start booking clinic appointments.

Login

Registered users can log in securely to manage their appointments.

Book Appointment

Students can choose an available doctor, date, and time slot to schedule their clinic visit.

Cancel Appointment

Students have the option to cancel appointments if they are unable to attend.

Admin Functions

Administrators can manage doctor availability and update schedules to ensure appointments run smoothly.

Contribution to SDG 3

This system supports SDG 3: Good Health and Well-Being by improving students’ access to healthcare services. By allowing online appointment booking and schedule management, it helps reduce long queues at the clinic and makes the healthcare process more convenient and efficient.
